## 2026-06-26 기준 상태

- 공개 명령은 `GSA4CHECK`, `GSA4GO`, `GSA4VERIFY`, `GSA4CLEAN`, `GSA4PDF` 다섯 개만 유지합니다.
- 기본 출력 설정은 `DWG To PDF.pc3`, `ISO full bleed A4 (297.00 x 210.00 MM)`, `monochrome.ctb`, prefix `SHEET`, plot rotation `0`입니다.
- A2/A3/A4처럼 프레임 크기가 섞인 경우에는 각 타입을 추가 샘플로 선택하고, 직접 선택한 샘플 타입은 일반 A계열 비율이 아니어도 같은 줄에서 비슷한 크기이면 포함합니다.
- 같은 줄 판정은 첫 프레임 Y 범위와 후보 프레임 세로 범위의 충분한 겹침을 기준으로 합니다.
- 배치 생성 시 뷰포트 잠금과 배치/뷰포트 내부 `GRIDMODE` 끄기를 유지합니다.
